# More cliff tiles, composition, and floating islands

**Goal:** Add more cliff tiles, and bigger ones. Make them compose together. Add (empty) tiles so we can have floating islands.

- Add more cliff tile variants (different shapes, heights) and larger cliff pieces.
- Design cliff composition (socketing or adjacency) so cliffs connect visually and logically.
- Add empty/void tiles (or “air” tiles) that allow gaps under level or cliff pieces so floating islands can form (supported by terrain rules rather than ground below).
