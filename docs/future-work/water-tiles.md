# Water tiles

**Goal:** Add water tiles as a type of ground tile. They should compose together similarly to level tiles.

- Define water as a ground tile type (tags, modules) that can sit on the 1.0 grid and connect via sockets.
- Water tiles should compose edge-to-edge (level-like composition) so ponds/lakes form coherent surfaces.
- Consider visual (water material, animation) and gameplay (collision, swimming/wading) in follow-up.
