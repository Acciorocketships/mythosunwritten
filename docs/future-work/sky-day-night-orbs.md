# Sky: day/night cycle and glowing orbs at night

**Goal:** Update the sky to cycle between night and day. At night, show glowing orbs floating around.

- Implement a day/night cycle (time of day driving sky colour/lighting).
- At night: reduce ambient light, tint sky, optionally show stars/moon.
- Add glowing orbs that float in the scene at night (particles or simple meshes with emissive material).
- Orbs should be visible only (or mainly) during night phase.
