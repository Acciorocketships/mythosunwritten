# Buildings in terrain generator

**Goal:** Add the buildings to the terrain generator. Animate the pieces in the buildings that need animating.

- Register building modules (or prefabs) with the terrain system so they can be placed by the generator (e.g. on level or ground tiles).
- Ensure buildings snap to grid and align with socket/placement rules.
- Identify and implement animations for building pieces that require them (e.g. doors, windmills, flags).
